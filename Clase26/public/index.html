Trabajamos la 2da forma de utilizar SASS


En esta segunda forma vamos a trabajar desde NODE. Para ello utilizaremos comandos que ya tenemos instalados, en este caso se utiliza en "npm", es el administrador que maneja NODE. Para crear un proyecto de SASS en NODE, se debe ejecutar el comando "npm init -y" para inicializar el proyecto y luego "npm install sass" para instalar SASS.

Desde la consola con NODE utilizamos el comando npm init para crear el archivo json que tendrá nuestro paquete a trabajar.

Si utilizo un npm init -y de esta manera nos creará un archivo package.json por default.

Una vez creado el archivo necesito agregarle herramientas para trabajar con SASS, para ello utilizo el comando npm install -g sass.

Al instalarlo descarga los paquetes en un directorio y además agrega la versión del soft en el archivo que habíamos creado.
