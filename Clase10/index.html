Seguimos trabajando la página de Messi

Vimos la clase pasada como trabajar con los Margin y con los Padding.

Recordemos que siempre podemos sacar todos los espaciados que aplica el navegador a nuestras páginas cuando utilizamos el selector * en nuestro archivo de CSS y le sacamos el Margin: 0 y Padding: 0 junto con Box-Sizing: border-box.

Estamos trabajando las opciones que nos trean las Fuentes Tipográficas. Importando estilos tipográficos desde páginas externas como google fonts, y para ello lo aplicamos a nuestro sitio de forma local y online.

El tercer método de uso de tipografía es el descargar la misma a nuestro directorio para luego llamarla. Desde el CSS en el inicio la llamamos con @font-face{ font-family: 'nombre'; src: url('../');} (crea el nombre y lo linkea a donde esta el archivo).
