Integramos BOOSTRAP y seguimos con Bucles

Bootstrap es una herramienta, un framework (plantillas de codigo predefinidas, es un marco de trabajo que no se deja manejar del todo, se integra a la web); REACT es una libreria compuesta por codigos de terceros, que puedo manipular. ANGULAR es también parecido a BOOTSTRAP, se deja manipular pero necesito cambiar mucho codigo para que funcione (son como cajas de codigo cerrado listo para ejecutarse).

Boostrap se utiliza en linea y se maneja desde CLASS es un archivo CSS lleno de código con clases para aplicar cada cambio. En su página web podemos encontrar ejemplos de como utilizar sus class y también tendremos una sección de docs en donde tenemos todo el contenidoque trae el CSS de Bootstrap.

Empezamos a aplicar las propiedades de FLEX dando formato a nuestras imágenes. Desde la ventana de inspección puedo ver en el CSS en la parte de FLEX un recuadro que me permite jugar o probar todas las propiedades de FLEX (alineaciones, espaciado, etc.).
