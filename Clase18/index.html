Seguimos con la clase de js

Completando la clase anterior de js, podemos meter opción 1 el script de js en el head como lo hariamos con css (esto hará que se ejecute ni bien arranca la página, no recomendable si tiene mucho código), la opción 2 es agregar el script al final del body (se ejecutará al finalizar de cargar la página, lo que es recomendable al tener mucho código en el archivo js), y la opción 3 es agregar el script como las etiquetas style dentro del cuerpo del html y escribir el código dentro de dichas etiquetas (se utiliza para casos específicos de algo que queramos modificar por código js en alguna parte el html).

Vemos como utilizar ventanas con js

Empezamos trabajando con la ventana de ALERT(), es una ventana de mensaje para que el usuario lo vea.

La segunda ventana que vemos es el PROMPT(), es una ventana para solicitar datos al usuario. Los mismos se pueden cargar a una variable para luego poder utilizar dicho valor.

La tercer ventana que vemos es el CONFIRM(), esta ventana nos muestra un mensaje y solo nos da la opción de aceptar o rechazar, esta ventana solo devuelve un valor TRUE o FALSE y sirve para tareas específicas cuando buscamos una respuesta de SI o NO.
