Seguimos con buclesy funciones


Funciones son pequeños boques de código que ejecutan una tarea. Dentro de estas funciones puedo crear variables que solo serviran para cuando se ejecute código dentro de la función, estas variables locales no existen fuera de la función. Puedo incluso darle varios parámetors a dichas funciones como si fueran un array.
