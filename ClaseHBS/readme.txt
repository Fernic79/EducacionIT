
En esta app instalamos las siguientes librerias:
npm i express path dotenv ecpress-handelbars mongoose morgan

Los .env contienen las variables de entorno

La libreria de Morgan me muestra en la consola cada una de las peticinoes que ocurren cuando se ejectua el front (vemos que responde con cada elemento de la pagina)

Utilice Bootstrap en el archivo principal de home.hbs que es donde voy a colocar mis plantillas para hacer la web

Los Partials son plantillas que apareceran en todas las plantillas (barra de navegacion, footer, etc).
