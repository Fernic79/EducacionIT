Mas comandos de GIT

Para bajarnos las carpetas de los repos donde trabajaremos debemos buscar el directorio que usaremos y clonar el repo GIT CLONE URL.

En este caso estamos compartiendo y agregando colaboradores, para ese caso necesitamos una serie de pasos:
Desde GITHUB cuando hacemos un repo podemos agregar colaboradores desde setting add colaboradores... buscamos los usauarios y los agregamos.
Recordar que cuando estemos trabajando en equipo debemos actualizar el repo de nuestro equipo antes de ponernos a trabajar, para ello utilizamos en la consola el GIT PULL esto nos actualiza los archivos de la nube a la pc.

Luego cuando estemos trabajando recordar GIT ADD . para actualizar los archivos a subir
Luego le hacemos un GIT COMMIT -M "" para empaquetar todo
Por ultimo solo resta subirlo GIT PUSH ORIGIN MASTER
